# 红魔队长 B 费 · Codex 桌面宠物

一个以红魔队长气质为灵感制作的 Codex v2 Q 版桌面宠物，包含经典摊手、指挥、庆祝小跳、复核以及 16 个指针朝向。

## 预览

上传项目文件后，可在此处查看动作总览：

![动作总览](contact-sheet.png)

## 特性

- 2D Q 版贴纸风，大头短身与赛璐璐阴影
- 红衣、白裤、黑袜及队长袖标
- 经典双掌朝上的摊手等待动作
- 九行标准动画
- 16 个连续指针朝向
- Codex v2 图集：1536 × 2288，8 × 11 网格
- 透明 WebP 图集，无透明 RGB 残留

## 文件

```text
pet.json              宠物配置
spritesheet.webp      Codex v2 动画图集
contact-sheet.png     全部动作总览
look-directions.png   16 个方向检查图
previews/             动作预览
```

## 安装

1. 下载仓库文件。
2. 将整个项目目录复制到：

   ```text
   %USERPROFILE%\.codex\pets\red-devil-captain-bruno
   ```

3. 确认目录中至少包含 `pet.json` 和 `spritesheet.webp`。
4. 重新打开 Codex，在宠物选择器中选择“红魔队长B费”。

## 配置

```json
{
  "id": "red-devil-captain-bruno",
  "displayName": "红魔队长B费",
  "description": "会指挥、会挑眉，也会用经典摊手等待你回应的红魔队长。",
  "spriteVersionNumber": 2,
  "spritesheetPath": "spritesheet.webp"
}
```
